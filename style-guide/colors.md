# Color Tokens

## Quick Reference

| Token | Hex Value |
| -- | -- |
| `primary` | `#0A2E22` |
| `secondary` | `#045C4E` |
| `accent` | `#E1F28F` |
| `surface` | `#f5ffef` |

## JSON Format

```json
{
  "colors": {
    "primary": "#0A2E22",
    "secondary": "#045C4E",
    "accent": "#E1F28F",
    "surface": "#f5ffef",
    "white": "#FFFFFF",
    "transparent": "transparent"
  }
}
```

## CSS Variables (If needed)

```css
:root {
  --color-primary: #0A2E22;
  --color-secondary: #045C4E;
  --color-accent: #E1F28F;
  --color-surface: #f5ffef;
}
```
