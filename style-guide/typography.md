# Typography System

**Font Family**: Inter (System Default)

## Scale

| Size Class | CSS | Line Height | Usage |
| -- | -- | -- | -- |
| `text-5xl` | `font-size: 3rem;` | `1` | Hero Layouts |
| `text-4xl` | `font-size: 2.25rem;` | `2.5rem` | Section Headers |
| `text-3xl` | `font-size: 1.875rem;` | `2.25rem` | Card Titles |
| `text-xl` | `font-size: 1.25rem;` | `1.75rem` | Lead Text |
| `text-lg` | `font-size: 1.125rem;` | `1.75rem` | Subtitles |
| `text-base` | `font-size: 1rem;` | `1.5rem` | Body Text |
| `text-sm` | `font-size: 0.875rem;` | `1.25rem` | UI Elements |
| `text-xs` | `font-size: 0.75rem;` | `1rem` | Captions/Tags |

## Weights
- **Extra Bold (800)**: Display headings.
- **Bold (700)**: Buttons, Navigation, Emphasized text.
- **Medium (500)**: Interactive links.
- **Regular (400)**: Body copy.
